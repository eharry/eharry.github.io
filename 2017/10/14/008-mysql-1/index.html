mysql 简介

  • mysql 登录: mysql -u root -p

  • mysql 登出:

  • 查看数据库: SHOW DATABASES;

  • 查看数据表: SHOW TABLES;

  • 显示列: SHOW COLUMNS FROM table_name;

  • 查看表结构: SHOW table, 这个就是 SHOW COLUMNS FROM table的快捷方式

  • 其他SHOW语句指令

  • SHOW STATUS: 显示服务器状态信息
  • SHOW CREATE DATABASE & CREATE TABLE SHOW 显示数据库和表的创建sql
  • SHOW GRANTS, 用来显示授权用户的安全权限
  • SHOW ERRORS & SHOW WARINGS 用于显示故障和告警

  • help SHOW 可以显示所有SHOW相关的命令

  • 显示所有存储引擎: SHOW ENGINES;

  • 显示权限: SHOW PRIVILEGES;

  • 显示连接信息: SHOW PROCESSLIST;
